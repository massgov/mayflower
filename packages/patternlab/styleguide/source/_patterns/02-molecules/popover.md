### Description
This pattern show a inline modal with helpful information.

### Status
* New as of 13.3.0

### Variables
~~~
tooltip: {
  controlId:
    type: string (unique per page) / required,
  openText:
    type: string / required
  closeText:
    type: string / required,
  title:
    type: string / optional,
  level
    type: number / optional
  message:
    type rich text / required.
}
~~~
