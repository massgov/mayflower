# Typography


## The official typeface of Mayflower is **Noto Sans**

Noto Sans was chosen because it's...
- **Accessible** and easy-to-read for users with all levels of language proficiency
- **Free** and Open-source so agencies can produce print and digital design, without barriers
- **Multilingual** to cover the languages our constituents speak and read
- **Modern** and consistent with the principles of the Mass.gov brand

Using these fonts for your print and digital assets is a subtle but compelling way to create a voice that is cohesive and unified with the rest of the Commonwealth digital ecosystem, and to ensure the text is accessible and easy to read.

> Noto Sans is an open source typeface jointly designed and developed by Google and Monotype. It has the most comprehensive language coverage — including more than 800 languages and 100 written scripts. You can directly download the fonts from Google Font, or for web projects, link to the fonts via Mayflower Assets CDN

## Styles and Usage

Noto Sans is used for all headings and body text. If your HTML is [semantic](https://developer.mozilla.org/en-US/docs/Glossary/Semantics) and uses tags that match the hierarchy of the content on a page, the Mayflower defaults will automatically style your headings with an appropriate amount of contrast.

<div className="reset-vspace">
  { Object.keys(headings).map((key, keyIndex) => (
      (key === "Body text") ?
      (<p>{key}</p>) :
      (<Heading level={headings[key]} key={`element.key.${keyIndex}`} text={key} />)
    )
  )}
</div>

### CSS weight classes

<p>Mayflower uses 5 custom weights of Noto Sans: <br /><strong>Lighter</strong> (250); <strong>Light</strong>, (350); <strong>Normal</strong>, (400); <strong>Bold</strong>, (550); and <strong>Bolder</strong>, (650).</p>

<p>Use the following classes to style text with those weights:</p>

<div className="sb-grid">

    styles.map((style, styleIndex) => {

        <React.Fragment key={`style.block.${styleIndex}`}>
