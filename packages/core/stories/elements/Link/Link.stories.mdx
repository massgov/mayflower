# Links

Links are used to embed actions or pathways to more information in a sentence or on a page.

### Inline Link

- **Usage:** Use when you are linking just a few words of text or when you are doing inline links.
- **Style:** Inline links take the style of the surrounding text, adding only the blue color and an underline.

### Decorative Link

- **Usage:** Used for standalone links that require more emphasis, usually inside `ma__section-links`.
- **Style:** the decorative link is slightly larger in size and appended with an arrow icon.

### Callout Link

- **Usage:** Call to action links that require more emphasis.
- **Style:** The callout link has the style of a decorative link, but with a slightly larger font-size, and enlosed in a box.

- **Style:** Styled exactly the same as `ma__button`.

### Accessibility & Best Practices

The purpose of a link can be determined from the link text alone or from the text combined with programmatically determined content

- Avoid using language in links that provides no specific information or is repetitive like “read more”.
- Links that start a process like downloading files, opening your e-mail client, or calling a phone number should be clearly labeled for what they do.
