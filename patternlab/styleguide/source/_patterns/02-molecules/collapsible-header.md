### Description
This is pattern shows an accordion button with an optional icon.

### Status
* Stable as of 5.0.0

### Variables
~~~
collapsibleHeader: {
  title: 
    type: string / required
  icon: 
    type: string (path to icon) / optional
  level:
    type: number / optional,
  info: 
    type: string / optional
}
~~~
