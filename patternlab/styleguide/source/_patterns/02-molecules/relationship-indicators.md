### Description
This is a set of related links for Organization pages to inculded the Organization links and optional related links.

### Status
* Stable as of 5.0.0

### Notes
Primary: at least one tag required
Secondary: optional

### Variables
~~~
headerTags: {

  primary: [{
    href:
      type: string / required
    text:
      type: string / required
  }],
   secondary: [{
    href:
      type: string / optional
    text:
      type: string / optional
  }]
}
~~~
