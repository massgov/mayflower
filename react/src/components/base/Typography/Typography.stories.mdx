  const [weight, italic] = style.split(' ');
  return italic ? `${weight.toLowerCase()}--${italic.toLowerCase()}` : weight.toLowerCase();
};

# Typography

**The official typeface of Mayflower is Noto Sans. **

### Usage

Mayflower uses the the following font weight options of Noto Sans:

| Font weight | Specs |
| :--- | :--- |
| Lighter | 250 |
| Light | 350 |
| Normal | 400 |
| Bold | 550 |
| Bolder | 650 |
